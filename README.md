# Computer-Communications-Project
Website for managing the Movies booking at the Cinema

An asp.net mvc Cinema application.

This Project was created with asp.net MVC framework (only works on windows), html, css, javaScript, jQuery, SQL .

The project was tested only with visual studio.

Project Run (Windows):
  
clone this repository.  
open visual studio.  
go to file -> open -> Project\Solution  
select NetworkProject.sln  
run the program (press start in visual studio)  

Application includes 3 types of users (Admin, Registered User, Unregistered User), each user has different options.
